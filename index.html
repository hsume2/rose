rose by hsume2

A slick Ruby DSL for reporting.

Install

{% highlight bash %} $ gem install rose {% endhighlight %}

Usage

{% highlight ruby %} Rose.make(:worlds) do rows do column(:hello => "Hello") column(:world) end end class World < Struct.new(:hello, :world) end Rose(:worlds).bloom([World.new("Say", "what?")]).to_s +---------------+ | Hello | world | +---------------+ | Say | what? | +---------------+ {% endhighlight %} {% highlight ruby %} WIP {% endhighlight %}
